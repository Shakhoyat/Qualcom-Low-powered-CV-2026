# 2026 LPCVC Track 2 Overview

Source: https://lpcv.ai/2026LPCVC/tracks/track2/

## Task
- Video Action Recognition.
- Input: 16-frame clip.
- Output: class prediction.

## Key Timeline
- Submission window: March 1, 2026 to April 30, 2026 (ET).

## Platform
- Hardware: Qualcomm Dragonwing IQ-9075 EVK.
- Software: Qualcomm AI Hub.

## Data
- Training data: open choice by participants.
- Sample data: QEVD.
- Hidden test set used by organizers.

## Submission Flow
- Compile and validate model on Qualcomm AI Hub.
- Share compile job/model access with lowpowervision@gmail.com.
- Fill official Track 2 submission form with compile job ID.
- Both steps are required for evaluation.

## Evaluation Summary
- Stage 1: execution time validity threshold.
- Stage 2: accuracy ranking among valid submissions.

## Notes
- Reference sample solution for preprocessing and expected tensor format.
- Keep checkpoint, ONNX, and compiled artifacts versioned by experiment ID.
